# 🚀 Remnanode VLESS Reality + Hysteria2 Helper

Подготовка ноды Remnawave для работы с **VLESS Reality + Hysteria2**.

---

# ⚡ Быстрая установка

```bash
bash <(curl -sSL https://raw.githubusercontent.com/Rrezzak09VPN/remnanode-VLESS-Reality-Hysteria2/main/setup.sh)
```

---

## 📦 Что делает скрипт

✅ Проверяет Docker и контейнер `remnanode`

✅ Проверяет сертификаты Let's Encrypt

✅ Копирует сертификаты в `/dev/shm`

✅ Настраивает автоматическое обновление сертификатов

✅ Создаёт Cron-задачи

✅ Перезапускает контейнер

---

## ⚠️ Важно

Скрипт **не изменяет конфиг ноды**.

Для генерации универсального конфига используйте конвертер:

🌐 https://rezzosoft.ru/converter.html

---

## 🔧 Порядок настройки

### 1. Подготовить сервер

Запустить команду установки.

### 2. Открыть ноду в Remnawave

Скопировать текущий JSON-конфиг ноды.

### 3. Открыть конвертер

🌐 https://rezzosoft.ru/converter.html

Вставить конфиг и получить новый **VLESS + Hysteria2**.

### 4. Загрузить новый конфиг в Remnawave

Создать новый Config и вставить туда результат конвертации.

### 5. Привязать конфиг к ноде

Выбрать новый Config в настройках ноды.

### 6. Создать хосты

* VLESS
* Hysteria2

### 7. Добавить inbound'ы в Squad

Добавить оба inbound в нужный Squad.

### 8. Готово 🎉

Нода будет одновременно работать через:

🚀 VLESS Reality

⚡ Hysteria2

---

## 🔄 Идемпотентность

Скрипт безопасно запускать повторно.

Он не создаёт дубликаты Cron-задач и не копирует сертификаты без необходимости.
