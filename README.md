# 🚀 Remnanode VLESS Reality + Hysteria2 Helper

Автоматическая подготовка ноды Remnawave для работы с конфигурациями **VLESS Reality + Hysteria2**.

Скрипт выполняет базовую подготовку сервера:

✅ Проверяет Docker и контейнер `remnanode`
✅ Проверяет наличие Let's Encrypt сертификатов
✅ Копирует сертификаты в `/dev/shm`
✅ Настраивает автоматическое обновление сертификатов через Cron
✅ Перезапускает контейнер после обновления сертификатов
✅ Проверяет базовые требования для дальнейшей работы Hysteria2

---
## ⚡ Быстрый запуск

Запуск одной командой:

```bash
bash <(curl -sSL https://raw.githubusercontent.com/Rrezzak09VPN/remnanode-VLESS-Reality-Hysteria2/main/setup.sh)
```

---


## ⚠️ Что делает этот проект

Этот проект **НЕ изменяет конфиг ноды в панели Remnawave**.

Скрипт занимается только серверной частью:

* сертификаты
* автоматизация
* подготовка окружения
* обслуживание после перезагрузки сервера

Конфигурация ноды (VLESS/Hysteria2) создаётся отдельно.

---

## 📋 Требования

Перед запуском должны быть выполнены следующие условия:

* Linux VPS
* Docker установлен и запущен
* контейнер `remnanode`
* рабочие сертификаты Let's Encrypt
* права root

---


## 🔍 Что происходит после запуска

Скрипт автоматически:

1. Проверяет окружение.
2. Находит сертификаты Let's Encrypt.
3. Копирует сертификаты в `/dev/shm`.
4. Создаёт cron-задачи:

   * после перезагрузки сервера;
   * ежедневно для обновления сертификатов.
5. Перезапускает контейнер `remnanode`.

---

## 🔄 Идемпотентность

Скрипт можно запускать многократно.

Он:

* не создаёт дубликаты cron-задач;
* не копирует сертификаты повторно без необходимости;
* безопасен для повторного запуска.

---

## 🛠 Использование вместе с конвертером конфигов

Рекомендуемая схема работы:

```text
Конфиг из панели Remnawave
            ↓
     Конвертер JSON
            ↓

            ↓
```
